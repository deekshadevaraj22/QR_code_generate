# 🔳 QR Code Generator (Text & Image)

This is a simple web application built using **Streamlit** that allows users to generate QR codes from:
- 📝 Text input  
- 🖼️ Image input  

---

## 🚀 Features

- Generate QR codes instantly
- Supports both text and image input
- Simple and user-friendly interface
- Built with Python and Streamlit
- Easy deployment on Streamlit Cloud

---

## 🛠️ Technologies Used

- Python
- Streamlit
- qrcode
- Pillow (PIL)
